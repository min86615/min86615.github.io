# Minhsuan Cheng — Personal AI Builder Portfolio

Static GitHub Pages-ready portfolio site.

## Publish with GitHub Pages

1. Create a repository named `<your-github-username>.github.io`.
2. Upload `index.html` and `styles.css` from this folder to the repository root.
3. In the repository, open **Settings → Pages**.
4. Select **Deploy from a branch**, choose `main`, and select `/ (root)`.
5. Add an email link later if you want direct email contact; the current CTAs point to LinkedIn.

The site uses Google Fonts through a CSS import. It otherwise has no build step or dependency.
